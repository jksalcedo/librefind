# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Added
- Support for pre-release updates (alpha, beta, RC) in the auto-updater.
- User preference to opt-in to experimental updates.
- Modernized Settings screen with a clean, list-based layout using reusable preference composables.

### Changed
- Refactored Settings UI to remove Card-based sections in favor of a standard Android settings aesthetic.
- Improved Settings ViewModel to better handle authentication state and user preferences.
